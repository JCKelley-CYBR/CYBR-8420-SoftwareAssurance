* Assurance Claim 3 - [The system ensures shared credentials confidentiality](https://github.com/JCKelley-CYBR/CYBR-8420-SoftwareAssurance/edit/Adding-Claims-Remaining/AssuranceCases/Credential_Confidentiality)
*Overview:* KeepassXC takes many steps to protect the confidentiality of information stored in the root databases, as well as in subsequent database shares. AES encryption is used for both encryption of the database. AES, or Argon2, is used for the key derivation function which generates a key from the users input password and stretches that key in accordance with secret key generation best practices.
